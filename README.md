# BinanceXI Site

Vite + React + TypeScript + Tailwind (shadcn-ui).

## Local development

```sh
npm i

npm run dev
```

## Production build

```sh
npm run build
npm run preview
```

## Deploy to Vercel

This repo includes `vercel.json` so React Router routes work on refresh (SPA fallback to `index.html`).

In Vercel:

- Import the Git repo
- Framework preset: `Vite`
- Build command: `npm run build`
- Output directory: `dist`
